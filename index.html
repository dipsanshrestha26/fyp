Sewa Ghar – Community Support Platform

Final Year Project (FYP) by Dipshan Shrestha

Project Overview

Sewa Ghar is a web-based platform designed to connect volunteers, donors, and people in need within local communities in Nepal. The system allows users to post requests for help (medical, education, food, shelter), donate resources or money, and track the impact of contributions in real time. It aims to make social service more organized, transparent, and accessible, especially in rural and urban areas of Gandaki Pradesh and beyond.

Objectives

  • Provide a centralized platform for posting and fulfilling community help requests
  • Enable secure and transparent donation tracking
  • Facilitate volunteer registration and task assignment
  • Raise awareness and encourage participation in social welfare activities

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: PHP with Laravel Framework
  • Database: MySQL
  • Deployment: GitHub Pages (frontend), shared hosting or VPS for backend
  • Additional Tools: Bootstrap for responsive design, Git for version control
