“SewaGhar” – Local Service Booking System

Final Year Project (FYP) by Dipsan Shrestha

Project Overview

“SewaGhar” is a web-based platform designed to connect customers with local service providers in Nepal for everyday services such as plumbing, electrical work, and more. The system allows users to search for services, make bookings, process payments, and leave reviews. It addresses challenges in the manual and unorganized service booking process by providing a centralized, digital solution that ensures reliability, transparency, and efficiency for customers, service providers, and admins.

Objectives

  • Develop a user-friendly platform for discovering and booking local services
  • Enable service providers to manage bookings, availability, and customer interactions
  • Provide admins with oversight tools for users, services, and platform operations
  • Incorporate secure payment gateways and user feedback systems to build trust and quality
  • Ensure accessibility and ease of use via any web browser

Technologies Used

  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Backend: PHP with Laravel Framework
  • Database: MySQL
  • Tools: Visual Studio Code, XAMPP, GitHub, Payment Gateway Integration, Google Maps API
  • Methodology: Iterative Development
